 備考
===================================================================
・WinMUGEN専用、Dos版では動作しません。

・グルーヴセレクト時にスタートボタンでEXグルーヴを選択することができます。

・MUGEN仕様の空中受身がどのグルーヴでも使用可能、
　時間差起き上がり、ガードクラッシュ、気絶はありません。

・config.txtの数値部分を変更することでEXグルーヴ他をカスタマイズすることが出来ます。

・特殊やられ対応
　　崩れダウン
　　投げ抜け
  　各種ガードクラッシュ
